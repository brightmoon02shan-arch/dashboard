---
name: table-info-metrics
description: >
  HR 指标定义速查表（Metrics）。定义所有标准化指标的公式、强制过滤条件和格式。
  包含原子指标（员工数/正式员工数/离职人次等）和复合指标（离职率/流失率/正式占比等），
  以及预入职/预离职/预转岗相关指标（#97-#111）。
  编写聚合 SQL 时参考本文件确认指标口径和过滤条件。
  ⚠️ 注意：#30/#31 的 new_flg 字段在 emp_df schema 中不存在（已废弃或占位），
  实际查询新员工应使用 wrk_age_mi_y>=0 AND wrk_age_mi_y<=1（司龄不满1年）。
  ⚠️ <yesterday_yyyymmdd_int> 是占位符，实际 SQL 中必须用 (SELECT MAX(date) FROM 对应表)。
  ⚠️ #39/#40 的闰年公式使用 Hive 语法（date_format/if），PostgreSQL 需改用 SUBSTRING/CASE。
